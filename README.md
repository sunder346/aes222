Aes шифратор\дешифратор

Простая прога на Python для шифрования и расшифровки текста. Алгоритм AES-256-CBC, интерфейс на Tkinter. 

Как запустить

Поставь Python (3.7+)
Установи либу: pip install cryptography
Запусти: python main.py

Как юзать
Вводишь пароль (минимум 4 символа)
Пишешь текст  жмёшь Зашифровать 
Чтобы расшифровать: вводишь тот же пароль, вставляешь полученную строку  Дешифровать 
Кнопка Результат → Вход перекидывает текст обратно, чтобы быстро проверить

Что внутри
AES-256-CBC
Ключ из пароля через PBKDF2 (200к итераций)
Соль + IV генерятся случайно при каждом шифровании
Итог кодируется в Base64, чтобы удобно копировать
